# -kipu-bank
Descripción del contrato.

Este contrato crea una boveda donde distintas cuantas puede depositar Ether con un límite maximo de depósitos y un limite máximo de extracción por transacción.

Acciones del contrato:
Depositar en la boveda
Retirar de la Boveda
Consutar saldo de una cuenta
Consultar cantidad de Depositos
Consultar cantidad de Retiros
Consultar cantidad de ETH acumulado en los depositos

Instrucciones de despliegue.
1- Definir Limite Depositos
2- Definir el Límite de retiros x Transacción
3- Desplegar el contrato

Cómo interactuar con el contrato.
1- desplegar el contrato en la cuenta 0
2- desde la cuenta 1 ingresar el Value (cantidad de ether en wei) Depositar
3- desde ll cuenta 1 definir la cantidad de ether a Retirar
4- comprobar saldos
